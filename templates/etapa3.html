Etapa 3 — Minería de Datos

En esta etapa se aplicaron técnicas de minería de datos utilizando Orange Data Mining con el objetivo de descubrir patrones ocultos, agrupamientos y relaciones no evidentes dentro del conjunto de datos de capacidad hospitalaria en Colombia.

Para ello se implementaron técnicas de clustering y reglas de asociación sobre el modelo multidimensional, permitiendo identificar perfiles hospitalarios, segmentaciones estructurales y relaciones recurrentes entre diferentes capacidades médicas.

Conclusiones Clustering

  • El clustering permitió identificar perfiles hospitalarios repetitivos dentro del sistema de salud colombiano.
  • Se detectaron agrupamientos asociados a hospitales de alta complejidad, caracterizados por capacidades como UCI, quirófanos y hospitalización avanzada.
  • Algunos grupos representan instituciones con capacidades médicas básicas, mientras otros concentran infraestructura especializada.
  • El análisis evidenció que ciertos servicios hospitalarios tienden a coexistir recurrentemente dentro de las mismas instituciones.
  • Se identificaron comportamientos atípicos asociados a configuraciones hospitalarias menos frecuentes.
  • Los resultados revelan que la distribución hospitalaria no es homogénea incluso dentro de un mismo departamento.

Conclusiones Reglas de Asociación

  • Las instituciones con capacidades medias de camas UCI para adultos presentan una fuerte tendencia a pertenecer a entidades de naturaleza jurídica tipo 2.
  • Las salas de observación con capacidades medias aparecen asociadas consistentemente a instituciones de naturaleza jurídica tipo 2.
  • La capacidad instalada de quirófanos modifica significativamente el perfil institucional hospitalario.
  • Algunos departamentos presentan patrones hospitalarios altamente homogéneos respecto a la naturaleza jurídica de las instituciones.
  • Las ambulancias básicas y servicios de atención inicial muestran relaciones recurrentes con capacidades hospitalarias reducidas.
  • El análisis permitió identificar una segmentación implícita del sistema hospitalario colombiano según la complejidad de la infraestructura médica.
