ANALISIS COMPARATIVO

Evolución del conocimiento

Esta sección resume la evolución del proceso analítico desarrollado durante el proyecto, mostrando cómo cada etapa permitió descubrir niveles progresivamente más profundos de conocimiento dentro del sistema hospitalario colombiano.

Tipo de conocimiento Características Resultado obtenido
Conocimiento evidente Análisis descriptivo básico mediante dashboards y estadísticas generales. Identificación de regiones, departamentos y capacidades hospitalarias con mayor volumen.
Conocimiento multidimensional Relación simultánea entre tiempo, ubicación, naturaleza jurídica y capacidad hospitalaria. Detección de comportamientos hospitalarios durante la pandemia y variaciones semestrales.
Conocimiento oculto Aplicación de clustering y minería de datos para encontrar patrones hospitalarios implícitos. Segmentación hospitalaria según complejidad médica y similitud estructural.
Conocimiento profundo Uso de procesamiento distribuido, machine learning y detección de anomalías hospitalarias. Descubrimiento de desequilibrios estructurales, crecimiento proyectado y concentración extrema de complejidad hospitalaria.
